Webhook EventTypes

body : { type: string, endpoint: string, name : string }

Values for types is listed below:

  • Payment Received. Object value : payment_received
  • Payment Received. Object value : payment_processing
  • Payment Received. Object value : payment_complete
  • Payment Received. Object value : payment

Webhook Registration Proceducre

  1. Via postman - Make a POST request to server endpoint /register
  2. In the request body you should have certain eventTypes you want to register with an endpoint URL and name of the webhook.
  3. The response from the server would be 202 - Accepted. This should be a good indication of the webhook working as it should
  4. If the eventType doesn't exist, 500 - Internal Server Errors. Due to the missing eventType, that is trying to be registered.
  5. When the Webhook has been initiated - then the server will try ping the endpoints of the remote servers.
